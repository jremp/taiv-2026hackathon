# Marketing and Design

In order to grow into a large company, you need to let your target audience know that you exist, and entice them to buy your product.

The goal of this module is to produce some visual aspect of your company that the judges can evaluate. Judges will be evaluating based on visual appeal and product understanding and cohesion. A couple examples of this are:

1. A logo for your company along with a writeup of your business objective and target demographic
2. A website home page that explains your product to possible users

## Installation Instructions

This web application was built using **Vite** and **Tailwind CSS v4**. Due to the bleeding-edge nature of the development environment (Vite 8), please use the following commands to ensure a clean build:

### 1. Install Dependencies

Run this command to bypass peer dependency conflicts between Vite 8 and the Tailwind plugin:

```bash
npm install --legacy-peer-deps

```

### 2. Launch the Application

Start the development server as recommended in the challenge documentation:

```bash
npm start

```

### 3. View the App

Once the server is running, open your browser to:
**[http://localhost:5173](http://localhost:5173)**
